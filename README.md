# 반려견 생활 정보 웹 애플리케이션

React와 REST API 기반 웹 애플리케이션 프로젝트

---

# 1. 분석/요구

## 1.1 프로젝트 배경
최근 반려견을 키우는 인구가 증가하면서 동물병원, 애견 미용실 등 반려견 관련 정보를 찾는 수요가 증가하고 있다.
하지만 이러한 정보는 여러 웹사이트에 흩어져 있어 사용자가 원하는 정보를 찾기 위해 많은 검색 과정을 거쳐야 한다.

따라서 반려견 관련 생활 정보를 한 곳에서 확인할 수 있는 웹 애플리케이션을 개발하고자 한다.

## 1.2 프로젝트 목표

* React 기반의 사용자 인터페이스 구현
* REST API 기반 데이터 제공 구조
* 반려견 관련 생활 정보 제공 웹 애플리케이션 개발
* GitHub를 이용한 프로젝트 코드 관리 및 문서화

## 1.3 주요 사용자

* 반려견 보호자
* 반려견 관련 정보를 찾는 사용자

## 1.4 제공 서비스

* 동물병원 정보 조회
* 애견 미용실 정보 조회
* 사용자 리뷰 기능

---

# 2. 설계 

## 2.1 시스템 구조

사용자는 브라우저로 React 기반 웹 페이지에 접속하며,
웹 페이지는 REST API 서버에 요청을 보내 관련 데이터를 받아 화면에 표시한다.

```
사용자 (웹 브라우저)
        ↓
React Frontend
        ↓
REST API Server
        ↓
데이터 (동물병원 / 미용실 정보)
```

## 2.2 주요 기능

1. 동물병원 정보 조회
2. 애견 미용실 정보 조회
3. 사용자 리뷰 작성

## 2.3 REST API 설계

### 동물병원 목록 조회

```
GET /api/hospitals
```

응답 예시

```
[
 {
  "name": "행복동물병원",
  "location": "경주",
  "phone": "054-123-4567"
 }
]
```

### 애견 미용실 조회

```
GET /api/grooming
```

### 리뷰 작성

```
POST /api/review
```

# 4. 프로젝트 보고서

## 4.1 프로젝트 개요

본 프로젝트는 React와 REST API 기반의 웹 애플리케이션 구조를 이해하고 구현하는 것을 목표로 한다.
반려견 관련 생활 정보를 제공하는 웹 애플리케이션을 개발하여 사용자가 동물병원이나 애견 미용실 정보를 쉽게 확인할 수 있도록 한다.

## 4.2 기대 효과

* React 기반 웹 애플리케이션 구조 이해
* REST API 기반 데이터 통신 이해
* GitHub를 활용한 프로젝트 관리 경험

## 4.3 향후 확장 계획

향후에는 IoT 센서를 활용하여 반려견의 활동량이나 위치 정보를 확인할 수 있는 반려견 관리 시스템으로 확장할 계획이다.

예시 확장 기능

* 반려견 활동량 측정
* 반려견 위치 추적
* 스마트 반려견 관리 시스템
